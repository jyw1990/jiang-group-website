# Jiang Group Website

Next.js static site for the Jiang Group at the University of Pennsylvania.

## Development

```bash
npm install
npm run dev      # http://localhost:3000
npm test         # run all tests
npm run build    # build static site to ./out
```

## Updating content

See `CLAUDE.md` for instructions on how to update news, people, research, publications, and photos.

## Deployment

Pushes to `main` automatically run tests and deploy to GitHub Pages via GitHub Actions.
